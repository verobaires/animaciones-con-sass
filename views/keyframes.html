Keyframe

Los @keyframes definen una animación en varios pasos. Permiten decirle al navegador cómo debe cambiar el estilo de un elemento a lo largo del tiempo.

              
              @keyframes nombre {  
                0% {  
                  /* estilos al comenzar */
                }  
                100% {  
                  /* estilos al finalizar */
                }
              }

¿Qué significan los porcentajes?

Los porcentajes indican el punto del tiempo total de la animación en el que deben aplicarse ciertos estilos.

¿Por qué usar varios porcentajes?

Si ponés solo uno, como 25% { ... }, se ejecuta ese cambio solo en ese punto. Si usás un rango, como de 0% a 100%, podés crear una animación completa y fluida. Los porcentajes sirven para controlar la progresión del movimiento.

Como aplicar el keyframe?

.caja {animation: mover 2s ease-in-out 0s infinite; }

animation: name duration timing-function delay iteration-count;

Valores dentro de la propiedad animation

Propiedad Que hace? Ejemplo Valores posibles
name Nombre del @Keyframes mover cualquier nombre
duration Duracion de la animacion 2s s(seg), ms (miliseg)
timing-function Como se distribuye la velocidad - Curva de aceleracion ease linear, ease, ease-in, ease-out, ease-in-out, cubic-bezier(...), steps()
iteration-count Cuantas veces se repite infinite 1, 2, infinite, 3.5
